Exporting case data from FA


WARNING: This has been reconfigured to accept SQL exports from FA (i.e., changed column headers). Management Reports from FA will NOT succeed.

  • Open Forensic Advantage.
  • Click on Management Reports in the left-hand sidebar.
  • Double-click SFME Case Analysis Report.
  • Set the Start Date and End Date. NOTE: Try to use a range of 6 months or less otherwise it will error out.
  • Click OK. Once it is done, it will open up a separate window and display a PDF.
  • In the top toolbar, click Export To and select CSV File.
  • Click 'OK' in the pop-up CSV Export Options.
  • Save the file to a local location.
  • Select that file for import using the form above.
